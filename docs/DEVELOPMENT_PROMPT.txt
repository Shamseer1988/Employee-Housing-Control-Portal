Architecture: Python Flask + PostgreSQL + SQLAlchemy + Alembic + Next.js + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion

====================================================================
PROJECT NAME
====================================================================

PUG Accommodation Control System

Build a centralized, advanced, responsive Employee Accommodation Management Web Application for a group of companies with multiple divisions, branches, properties, employee accommodations, landlords, rooms, and bed allocations.

This system must work as a professional internal control system like the existing Finance App architecture, but with a modern responsive frontend.

====================================================================
BUSINESS CONTEXT
====================================================================

We are a group of companies with multiple branches and divisions in different locations. Employees are accommodated in multiple properties such as full buildings, partial buildings, one floor only, villas, apartments, rooms, and shared bed spaces.

All accommodation control is handled centrally from Head Office.

Employees from different divisions can stay in the same property or room area. The system must provide complete visibility, control, audit trail, reports, reminders, and user privilege restrictions.
